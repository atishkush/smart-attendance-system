# 🎯 Smart Attendance System

### Face Recognition Based Attendance Management

A computer vision powered attendance tracking system that automatically identifies registered users and records attendance using facial recognition technology.

</div>

---

## 📌 Overview

Smart Attendance System is designed to simplify attendance management through automated face recognition. The application detects faces from a live video stream, verifies user identity, and records attendance in real time.

This project demonstrates the practical application of Machine Learning and Computer Vision in automating routine administrative tasks.

---

## ✨ Features

* Real-Time Face Detection
* Face Recognition Authentication
* Automated Attendance Recording
* Live Video Feed Monitoring
* User Registration Management
* Attendance History Tracking
* Web-Based Dashboard

## 🚀 Installation

### Backend Setup

```bash
cd backend
conda env create -f environment.yml
conda activate attendance-system
```

### Frontend Setup

```bash
cd frontend
npm install
```

---

## ▶️ Running the Project

### Start Backend

```bash
python run.py
```

### Start Frontend

```bash
ng serve -o
```

Application URL:

```text
http://localhost:4200
```
